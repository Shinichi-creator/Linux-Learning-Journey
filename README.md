# Linux-Learning-Journey

This repository documents my journey learning Linux from the ground up as part of my cybersecurity roadmap. I am a first- year cybersecurity major at purdue Global, building Linux, networking, and system administration skills to support future work in cybersecurity and cloud.

Repository Structure
LinuxJourney-Notes/ – notes from https://linuxjourney.com/ and other learning.
Commands-CheatSheets/ – quick-reference command lists.
System-Administration/ – users, groups, services, permissions, logs, etc.
SSH/ – SSH basics, keys, hardening, and troubleshooting.
Networking/ – Linux networking commands and notes.
Debugging/ – errors I hit and how I fixed them.
Scripts/ – bash scripts I write as I learn.
Arch-Install/ – notes and logs from my future Arch Linux install.

Goals
Get comfortable using Linux from the terminal.
Understand users, groups, and permissions.
Learn basic networking and troubleshooting on Linux.
Practice scripting and automation with bash.
Prepare for future certifications and cybersecurity work.
